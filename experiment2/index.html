Experiment 2 - Vector Art, Animation & Interactivity

Description

In this experiment, I wanted to added mouse interactivity to an existing project of nodes. The nodes are repelled by the mouse, but are attracted when left clicking.

Technical

The main features, mouse repelling and attracting, works by computing a force vector and adding it to the node's velocity. This force vector is determined using the distance between the mouse and node and whether the mouse is pressed or not. All this functionality is put within a function in Node.js, which is called in sketch.js.
