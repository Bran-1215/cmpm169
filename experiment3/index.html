Experiment 3: Generative Methods

Description

With this experiment, I wanted to simulate the growth of slime mold using cellular automata, but I ended up with something more akin to gray goo.

Technical

This experiment simulates slime mold behavior using a grid-based cellular automata system. The canvas is divided into cells, each representing slime density, chemical signals, or food, with growth and decay driven by rules influenced by chemical gradients and randomness. Slime molds grow toward food sources, decay when overcrowded, and respawn in a central region if no slime remains. Interactive features include a 2x2 erase tool triggered by mouse input.
